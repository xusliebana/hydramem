# Product Hunt

**NAME:** HydraMem
**TAGLINE (≤60 chars):** `Local, private long-term memory for your AI agent`
**TOPICS:** Artificial Intelligence, Developer Tools, Open Source, Privacy

**DESCRIPTION:**
```
HydraMem gives your AI coding agent (Claude Desktop, Cursor, OpenCode) persistent
long-term memory that runs 100% on your machine. Hybrid graph + vector search,
two-stage verification so it never stores hallucinated facts, and an offline
"Night Gardener" that refines the knowledge graph overnight. ~70% token savings vs
naive RAG — and every number is auditable. MIT, ~5k LOC, MCP-native.
```

**FIRST COMMENT (maker):**
```
Hi PH 👋 I built HydraMem because I wanted my agents to remember things across
sessions without shipping my codebase context to a cloud memory service.
It's deliberately honest: it emits zero relations when there's no evidence, and
`hydramem stats --raw` audits exactly what it saved. It's small enough to read in an
afternoon (~5k LOC) and MIT-licensed. Would love your feedback — especially edge cases
where local-first hurts. AMA!
```
