# BM25 Lexical Retrieval

BM25 is the keyword arm of HydraMem's hybrid retrieval. It scores documents by
Okapi term frequency with length normalisation, surfacing exact keyword and
identifier matches that dense embeddings often miss. The BM25 ranking is fused
with the vector and graph rankings through Reciprocal Rank Fusion so lexical and
semantic signals reinforce each other.
